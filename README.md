# PSYC 7304/8304: Measurement Theory & Psychometrics — Course Resources

Supplementary slides, simulated datasets, and reference guides organized by course module. Assignment instructions and lab submissions remain on Canvas. Some content was developed with AI assistance and reviewed by the instructor. Licensed under MPL 2.0.

## Contents

Materials are organized by module under `/modules`. Each folder contains rendered lecture slides, simulated datasets with codebooks, and supplementary guides where relevant.

## Usage

The student-facing landing page is available at the link provided in your course syllabus. Navigate from there — do not rely on the folder structure in this repository directly.

## Versioning

This repository is tagged by course version (e.g., `v2026`). The current state of `main` reflects the active semester. Prior versions are archived via tags.
